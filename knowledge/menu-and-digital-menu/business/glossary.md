# Glossary - Fararest Menu Module

## Core Terms

| Term | Meaning |
|---|---|
| Menu Management | ماژول مدیریت ساختار منو، آیتم‌ها، قیمت، موجودی، کانال‌ها، شعب و قوانین فروش |
| Digital Menu | نسخه وب/موبایل منو که از طریق QR، لینک، سایت، پیامک یا شبکه اجتماعی در دسترس مشتری است |
| Kiosk Menu | نسخه لمسی یا نمایشگری منو برای کیوسک، IPTV، Digital Signage یا دستگاه سلف‌سرویس |
| Display Mode | حالت نمایشی بدون تعامل جدی، مناسب نمایش قیمت، آیتم و کمپین |
| Interactive Mode | حالت تعاملی برای جستجو، انتخاب، شخصی‌سازی، سفارش، پرداخت و رسید |
| Item | هر چیزی که قابل فروش یا انتخاب است؛ مانند غذا، نوشیدنی، محصول شرکتی، سرویس، پکیج، کمبو یا افزودنی |
| Produced Item | آیتمی که از مواد اولیه یا رسپی تولید می‌شود |
| Corporate/Packaged Product | محصول آماده یا شرکتی که الزاماً رسپی تولیدی ندارد |
| Modifier / Option | انتخابی که روی آیتم اعمال می‌شود؛ مانند سایز، نوع نان، سطح پخت، حذف ماده یا افزودنی |
| Add-on / Topping | مورد اضافه قابل فروش روی آیتم اصلی |
| Combo / Bundle | ترکیبی از چند آیتم با قیمت یا منطق فروش مشخص |
| Channel | محل یا روش فروش؛ مانند سالن، بیرون‌بر، آنلاین، پیک، سازمانی، کیوسک یا پلتفرم ثالث |
| Branch Override | استثنای شعبه‌ای نسبت به منوی مرکزی |
| Availability | قابل فروش بودن آیتم بر اساس موجودی، زمان، شعبه، کانال یا وضعیت دستی |
| Scheduled Menu | منویی که بر اساس ساعت، روز، شیفت، فصل یا مناسبت فعال می‌شود |
| Digital Signage | نمایشگر عمومی برای نمایش غیرتعاملی آیتم‌ها، قیمت‌ها، باندل‌ها و کمپین‌ها |
| Customer Journey Drop-off | مرحله‌ای که مشتری در آن مسیر مشاهده یا سفارش را رها می‌کند |
| Menu Source of Truth | منوی مرجع واحد که سایر کانال‌ها از آن تغذیه می‌شوند |
