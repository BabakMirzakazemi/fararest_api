# Frontend Screens - Fararest Menu Module

## Menu Management Screens

### 1. Menu Dashboard

Purpose:

- quick operational overview
- important warnings
- shortcut actions

Primary actions:

- add item
- add category
- bulk price update
- temporarily disable items
- copy menu to another branch
- publish/preview menu

### 2. Category Management

Features:

- multi-level category tree
- drag-and-drop sorting
- image/color/description settings
- active/inactive toggle
- prevent unsafe delete if active items exist

### 3. Item List

Features:

- card/table view switch
- instant search
- advanced filters
- inline edit
- bulk actions
- status indicators

### 4. Item Create/Edit Wizard

Steps:

1. Basic info
2. Pricing
3. Availability
4. Inventory/recipe
5. Options/modifiers
6. Channels/branches
7. Preview/confirm

### 5. Publishing & Versions

Features:

- draft preview
- scheduled publish
- version history
- rollback
- approval if required

### 6. Analytics Dashboard

Features:

- best/low selling items
- category contribution
- margin
- unavailable frequency
- item issues
- price change impact
