# Project Notes

## Feature جدید

این repository اکنون یک قابلیت `Episodic Memory` دارد که برای نگهداری durable project/event memory طراحی شده است.

## لایه‌بندی

- Domain entities: `Entities/EpisodicMemory`
- Application contracts/DTOs/services: `Services/**/EpisodicMemory`
- EF mapping: `Data/Configurations/EpisodicMemory`
- API endpoints: `API/Controllers/Admin/v1/EpisodesController.cs`

## هدف

این feature برای استفاده مستقیم عامل‌های هوش مصنوعی، تیم backend، و عملیات طراحی شده تا تجربه‌های مهم پروژه از بین نرود و دوباره‌کاری کاهش یابد.
